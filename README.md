Viikkotehtävä 1: Kotlin-perusteet ja Compose
Tässä tehtävässä harjoiteltiin Kotlin-kielen perusteita sekä Android-projektin käynnistämistä Jetpack Composella.

Mitä opittiin?
Luomaan yksinkertainen datamalli (data class).
Käsittelemään listoja Kotlinilla (lisäys, suodatus ja järjestäminen).
Rakentamaan ensimmäinen yksinkertainen käyttöliittymä Composella.


Logiikka: Toteutettiin funktiot tehtävälistan hallintaan:
addTask: Uuden tehtävän lisääminen.
toggleDone: Tehtävän tilan muuttaminen (tehty/tekemättä).
filterByDone: Tehtävien suodatus tilan perusteella.
sortByDueDate: Listan järjestäminen päivämäärän mukaan.
Käyttöliittymä: Tehtiin HomeScreen-näkymä, joka listaa mock-datan käyttäen Column- ja Text-komponentteja.
